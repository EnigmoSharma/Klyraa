Our Mission

Our goal is to demonstrate how IoT and real-time data can simplify parking management. This project integrates sensors, embedded systems, and web technologies to provide a prototype solution.

While still a prototype, the system shows how automation can make parking more efficient and user-friendly in a campus or local setting.

Key Features of the Klyra Smart Parking System

Features of our smart parking prototype

Parking Spot Detection

Uses IR sensors to detect whether a parking slot is occupied or free.

Real-time Updates

Slot availability is updated instantly on the web portal.

Simple Reservations

Users can reserve available slots through the demo system.

Quick Slot Allocation

Slots are assigned immediately upon booking in the prototype.
